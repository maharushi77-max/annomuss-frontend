🔔Notification CenterCRITICAL
Bell icon exists but clicking it does nothing. Users get friend requests, likes, messages — and have no idea. This breaks the core social loop.
→ Create notifications.html. Show: new messages, friend requests, post likes, challenge responses. Store in DB with a "read" flag.
✓
🔍Search Users & PostsCRITICAL
If someone tells a friend "find me on Annomuss as SilentMoon99", there's no way to find them. No hashtag search. No user lookup. Growth completely blocked.
→ Search bar in topbar. GET /api/users/search?q= already exists. Add GET /api/posts/search?q= for post search. Results page with tabs.
✓
📱Web Push NotificationsHIGH
Users close the app and never come back. Without push notifications, there's zero re-engagement. Someone likes your post at 11pm — you find out the next time you manually open the app.
→ Service Worker + Web Push API (free). Ask permission on first login. Send on: new message, friend request, like on your post, challenge response liked.
✓
👤Public Profile PagesHIGH
Tap a username → see their public posts, streak badge, XP level, bio. Currently it does nothing. No social identity = no reason to build one.
→ Create profile.html?user=anonymousName. GET /api/users/profile/:name. Show public posts, streak, level, bio, common friends.
✓
💬Pre-fill 30 Days of ChallengesHIGH
Challenge tab is empty every day until you manually create one. If you forget for a day — users see nothing. Pre-fill a month in advance.
→ Use challenge-seeder.html (in your outputs). One click fills 30 days with emotionally resonant prompts. Takes 2 minutes.
✓
📊XP Syncs to DatabaseHIGH
All XP is in localStorage. Clear browser = lose everything. Log in on phone = 0 XP. Users build up 500 XP then feel robbed when it vanishes. Kills motivation.
→ Deploy new users.js + User.js (already in outputs). PATCH /api/users/xp syncs on each gain. GET /api/users/xp loads on login.

🟢 Unique Features — What Makes Annomuss Unforgettable

Viral Potential
🌊
Weekly Emotional Report
Every Sunday: "This week, 847 people felt anxious, 312 felt hopeful, 156 said they missed someone..." A community emotional weather report that people screenshot and share everywhere.
→ Aggregate last 7 days of moods. Auto-post to feed every Sunday.
🕛
Midnight Mailbox
Between 11pm–3am, a special feed unlocks where posts auto-delete at dawn. Radical honesty because everything disappears. Scarcity + mystery = massive FOMO. People tell friends about this.
→ Add expiresAt = next 6am when type="night" + time is 23:00–03:00.
💌
Time Capsule Letters
Write a letter to yourself in 30/90/365 days. When it unlocks — it gets posted to the feed. The emotional payoff of reading your past self is deeply shareable content.
→ Add releaseAt field to Post. Check + publish on read. Show countdown.
🫂
Vibe Match
After logging a mood, show: "3 other people in your city felt anxious in the last hour — just like you." Not a chat, just a moment of "you're not alone." Deeply emotional feature.
→ After mood log, query users with same emotion in last 1h. Show count.
🌱
Emotional Growth Tree
A visual tree that grows with the user. Each mood log = leaf, each post = branch, each goal completed = flower. Screenshot-able identity. Makes streak feel alive, not just a number.
→ SVG animation based on user stats. Private art under profile page.
🎭
Persona Masks
Once a month, wear a different mask — a new anonymous identity for 24 hours. Complete reset. Write things you couldn't write as yourself. Makes venting safer and more honest.
→ Add maskName + maskExpiry to User. Show masked name during active window.
