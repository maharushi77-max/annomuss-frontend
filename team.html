Quick FAQ — Common Issues
OTP not received
Check spam folder. Email takes 1-2 min. Resend domain must be verified on resend.com/domains for new email IDs.
Posts not loading
Hard refresh (Ctrl+Shift+R). If issue persists check Render logs for server errors.
Messages not sending
Verify routes/messages.js is deployed. Check Render logs for "messages.js loaded (sync in start)".
Account banned wrongly
Go to Admin Panel → Users → find the user → Unban. User must re-login after unban.
